# 👑 로얄티 (Royalty)
> **"당신의 브랜드 가치를 지키는 AI 상표 감시 파트너"**

사용자가 로고와 상표명을 등록하면, AI가 유사한 상표가 있는지 분석하고 위험도를 알려주는 지식재산권(IP) 보호 서비스입니다.

## 🛠️ Tech Stack
- **Frontend:** React
- **Backend:** FastAPI
- **DB:** PostgreSQL, Azure Blob Storage
- **AI:** ResNet50 (Image), SBERT (Text)
- **Infra:** Azure DevOps, Docker

---

## 🚨 Git 협업 규칙 (Team Royalty Rule)

### 1. 브랜치 전략
- **main:** 🚀 배포용 (절대 건드리지 않기)
- **develop:** 🛠️ 개발용 (모든 기능은 여기로 합쳐짐)
- **feature/[기능명]:** 👨‍💻 각자 작업 공간

### 2. 커밋 메시지 컨벤션
- `[FE]`: 프론트엔드 작업
- `[BE]`: 백엔드 작업
- `[AI]`: 모델 및 데이터 작업
- `[DOCS]`: 문서 수정 (README, ERD 등)
